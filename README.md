# HW7

Pretty standard implementation of Pac-Man here. I've detailed what I've done so far, some bugs, and what features need to be added.

Features: Basic player and ghost movement logic, basic animations, pelets, win/lose graphic, map. Easily editable constants at the top of the file to tweak things like speeds, movement logic, colors, etc. 

Features still Needed:
    1. Power pellets to power up Pacman so he can eat ghosts
    2. Actual tracking logic for ghosts to try and get pacman out
    3. Exits on side of maps to switch sides
    4. Musics/Better animations for ghosts and pacman

Bugs:
    1. Game is a bit fast and choppy, can probably fix using constants
    2. Ghosts can overlap each other
    3. Some terminal issues, where you need to close the Python terminal when you close the game to be able to run it again
